# Gestfinance

O **GestFinance** é um projeto integrador desenvolvido com o objetivo de auxiliar usuários no **controle de finanças pessoais** de forma prática e acessível.  
A aplicação busca oferecer uma interface intuitiva para cadastro de contas a pagar e receber, além de exibir resumos financeiros que ajudam na organização do orçamento.
