# Gestfinance

O **GestFinance** é um projeto integrador desenvolvido com o objetivo de auxiliar usuários no **controle de finanças pessoais** de forma prática e acessível.  
A aplicação busca oferecer uma interface intuitiva para cadastro de contas a pagar e Ganhos, além de exibir resumos financeiros que ajudam na organização do orçamento.  


Mini WBS – Projeto GestFinance

### 1. Sobre
O projeto **GestFinance** tem como objetivo ajudar usuários a organizar suas finanças pessoais de forma simples e acessível, com funcionalidades de login, cadastro de contas e resumo financeiro.

---

### 2. Funcionalidades
- Login e cadastro de usuários  
- Registro de contas a pagar e ganhos  
- Exibição de totais (gastos, dívidas, sonhos, emergência)  
- Interface responsiva e amigável  

---

### 3. Quem vai fazer o quê
Todas as etapas serão realizadas em conjunto pela equipe, garantindo colaboração e divisão equilibrada das tarefas.

### 6. Dor e Público-Alvo
- **Dor:** dificuldade em organizar gastos e planejar objetivos financeiros  
- **Público-alvo:** jovens e adultos que desejam controlar suas finanças pessoais de forma prática e acessível
